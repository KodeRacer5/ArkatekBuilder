## 2. What a Partner Actually Gets

When a partner white-labels this system, they are receiving:

**Licensed (from ArkatekBuilder):**
- The DREAMS assessment framework and vertical taxonomy
- The DREAM Score composite calculation engine
- The CortixEngine prompt orchestration and data mapping layer
- The scoring ring UI system and opportunity sizing output
- The integration panel UI and connection flow

**Not licensed (commodity APIs — partner registers their own):**
- QuickBooks Online API credentials (Intuit Developer Program)
- Plaid API credentials (Plaid Dashboard)
- Gusto API credentials (Gusto Developer Portal)
- Stripe API credentials (Stripe Dashboard)
- Ramp, Salesforce, ADP, Xero, NetSuite — same, partner registers directly
