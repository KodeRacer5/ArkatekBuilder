Router Node
Diamond shape via 45deg CSS rotation. 4 connection handles (L/R/T/B), all counter-rotated -45deg. Content inside counter-rotated to stay upright. Used for SPLIT and MERGE junctions in the pipeline.
type keyrouter
size80×80px (fixed square, rotated)
transformrotate(45deg) on outer div
innerrotate(-45deg) to keep text upright
backgroundrgba(12,14,20,0.92)
border2px solid {color}88
handles4 — Left, Right, Top (id=top), Bottom (id=bottom)
handle offset-6px from edge (not -5px, allows for rotation)
icon font14px / 800 / {color}
label font9px / 700 / #fff
use caseSPLIT, MERGE, routing junctions
#06b6d4 cyan (SPLIT)
#a855f7 purple (MERGE)
